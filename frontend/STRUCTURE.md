/**
 * Frontend Structure Guide
 * 
 * /src
 *   /components      → Reusable UI components (Badge, Button, Modal, etc.)
 *   /pages           → Full page components (LoginPage, DashboardPage)
 *   /services        → API calls and external services
 *   /hooks           → Custom React hooks (useAuth, useWebSocket)
 *   /utils           → Utility functions and constants
 *   /styles          → Global CSS files
 *   App.jsx          → Main app component
 *   main.jsx         → Entry point
 * 
 * /public            → Static assets (images, fonts, etc.)
 * 
 * Key Files:
 * - services/api.js      → All API endpoints
 * - utils/constants.js   → Email types, colors, config
 * - hooks/useAuth.js     → Authentication logic
 * - hooks/useWebSocket.js → WebSocket connection
 */
